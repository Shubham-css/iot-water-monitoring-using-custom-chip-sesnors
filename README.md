# 💧 IoT Water Quality Monitoring System

Real-time IoT-based system to monitor water quality parameters using ESP32 and custom sensor integration, simulated on Wokwi.

---

## 🚀 Features

* 📡 Real-time monitoring of water parameters
* 🌡️ Temperature sensing
* 🧪 pH level detection
* 💧 TDS (Total Dissolved Solids) measurement
* 🌫️ Turbidity analysis
* ⚙️ Custom sensor chips simulation using Wokwi

---

## 🛠️ Tech Stack

* ESP32
* Arduino (Embedded C/C++)
* Wokwi Simulator
* IoT Sensors (pH, TDS, Turbidity, Temperature)

---

## 📂 Project Structure

```
.
├── diagram.json
├── sketch.ino
├── libraries.txt
├── *.chip.c / *.chip.json
└── wokwi.toml
```

---

## ▶️ How to Run

1. Open the project in Wokwi
2. Upload the `diagram.json` file
3. Run the simulation
4. Modify sensor values using sliders
